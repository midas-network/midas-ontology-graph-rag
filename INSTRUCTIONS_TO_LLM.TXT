## 1. Code Architecture

### 1.1 Function Design
- **Single responsibility**: Each function does one thing well
- **Small helpers**: Extract logic into focused helpers (< 25 lines preferred)
- **Pure functions**: Prefer functions without side effects; isolate I/O at boundaries
- **Explicit arguments**: No mutable default arguments; use `None` sentinel pattern:
  ```python
  def process(items: list[str] | None = None) -> list[str]:
      items = items if items is not None else []
  ```
- **Limit parameters**: Max 5 positional args; beyond that, use dataclasses/TypedDict or keyword-only (`*`)

### 1.2 Class Design
- Use classes for stateful components; prefer functions for stateless transformations
- Favor composition over inheritance
- Use `@dataclass` or `@dataclass(frozen=True)` for data containers
- Use `Protocol` for structural typing over ABCs when possible

### 1.3 Module Structure
- No executable code at module level (except `if __name__ == "__main__":`)
- Imports must not trigger side effects
- Keep modules import-safe: lightweight setup only

### 6.2 Patterns
```python
# Custom exceptions for domain errors
class LLMConnectionError(Exception):
    """LLM host unreachable."""
    pass

class ExtractionError(Exception):
    """Extraction pipeline failure."""
    pass

# Context manager for resource cleanup
from contextlib import contextmanager

@contextmanager
def managed_connection(host: str, timeout: int = 30):
    conn = None
    try:
        conn = connect(host, timeout=timeout)
        yield conn
    except ConnectionError as e:
        raise LLMConnectionError(
            f"Cannot reach LLM host: {host}\n"
            f"Check LLM_HOST env var or verify service is running"
        ) from e
    finally:
        if conn:
            conn.close()
```

### 6.3 Rules
- Never swallow exceptions silently
- Surface actionable hints: missing env vars, file paths, hosts, models
- Use exception chaining (`raise ... from e`)
- Log before raising if context would otherwise be lost

## 8. I/O and File Handling

### 8.1 Output file naming
```python
from datetime import datetime
from pathlib import Path

def get_output_path(
    base_dir: Path,
    prefix: str,
    suffix: str = ".json",
) -> Path:
    """Generate timestamped output path.
    
    Example: runs/extraction_20240115_143022.json
    """
    timestamp = datetime.now().strftime("%Y%m%d_%H%M%S")
    return base_dir / f"{prefix}_{timestamp}{suffix}"
```

### 8.2 Safe file writing
```python
import json
from pathlib import Path
from tempfile import NamedTemporaryFile

def write_json_atomic(path: Path, data: dict) -> None:
    """Write JSON atomically to prevent partial writes."""
    path.parent.mkdir(parents=True, exist_ok=True)
    
    with NamedTemporaryFile(
        mode="w",
        dir=path.parent,
        suffix=".tmp",
        delete=False,
    ) as f:
        json.dump(data, f, indent=2)
        temp_path = Path(f.name)
    
    temp_path.rename(path)
```

### 8.3 Capturing stdout/stderr

import subprocess
from pathlib import Path

def run_with_capture(
    cmd: list[str],
    log_path: Path,

) -> subprocess.CompletedProcess:

## 9. LLM Integration

### 9.1 Host probing
```python
import httpx

def probe_llm_host(host: str, *, timeout: int = 5) -> bool:
    """Check if LLM host is reachable."""
    try:
        response = httpx.get(f"{host}/health", timeout=timeout)
        return response.status_code == 200
    except httpx.RequestError:
        return False

def ensure_llm_available(host: str, model: str) -> None:
    """Validate LLM availability before processing."""
    if not probe_llm_host(host):
        raise LLMConnectionError(
            f"LLM host unreachable: {host}\n"
            f"Set LLM_HOST env var or start the service"
        )
```

### 9.2 Test mode
```python
def call_llm(
    prompt: str,
    *,
    host: str,
    model: str,
    timeout: int = 30,
    test_mode: bool = False,
) -> str:
    """Call LLM with optional test mode for validation."""
    if test_mode:
        test_prompt = "Respond with 'OK' if you can read this."
        # Use short timeout for probe
        response = _send_request(host, model, test_prompt, timeout=5)
        if "OK" not in response:
            raise LLMConnectionError(f"Test probe failed: {response[:100]}")
        return "[TEST MODE] LLM connection verified"
    
    return _send_request(host, model, prompt, timeout=timeout)
```

### 9.3 Rules
- Probe hosts before use
- Honor configured model/host from config/env
- Provide reasonable timeouts
- Offer TEST mode that sends a short probe message

## 11. Similarity and Matching

```python
from difflib import SequenceMatcher

def compute_similarity(
    a: str,
    b: str,
    *,
    use_embeddings: bool = True,
    embedding_model: Any | None = None,
) -> tuple[float, str]:
    """Compute string similarity, reporting method used.
    
    Returns:
        Tuple of (similarity_score, method_used)
    """
    if use_embeddings and embedding_model is not None:
        try:
            vec_a = embedding_model.encode(a)
            vec_b = embedding_model.encode(b)
            score = cosine_similarity(vec_a, vec_b)
            return score, "embeddings"
        except Exception:
            pass  # Fall through to difflib
    
    # Fallback to difflib
    score = SequenceMatcher(None, a.lower(), b.lower()).ratio()
    return score, "difflib"
```

        "count": len(values),

## 13. Prompt Engineering

### 13.1 Rules
- Keep instructions minimal and precise
- Avoid redundant reminder blocks unless toggled by flag
- Use numbered fields for multi-value attributes
- Store prompts in `data/prompt/` as templates

### 13.2 Template structure
```python
# data/prompt/extraction_base.py
EXTRACTION_PROMPT = """Extract the following attributes from the text.

For each attribute, provide:
1. value: The extracted value
2. provenance: Exact text span supporting this extraction  
3. reasoning: Brief explanation of extraction logic

{schema_block}

Text:
{input_text}

Output as JSON."""

def build_prompt(
    text: str,
    schema: dict,
    *,
    include_examples: bool = False,
) -> str:
    schema_block = format_schema(schema)
    prompt = EXTRACTION_PROMPT.format(
        schema_block=schema_block,
        input_text=text,
    )
    
    if include_examples:
        prompt = add_examples(prompt, schema)
    
    return prompt
```
